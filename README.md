# CalibraPy

***CalibraPy*** é uma ferramenta de calibração de sensores, utilizando placa de aquisição Arduino. Através de um ensaio
estático e um ensaio de resposta ao degrau, é possível caracterizar tanto a resposta dinâmica quanto a resposta estática
de um sensor.

Para utilizar o ***CalibraPy***, siga os passos abaixo:

- Faça o *download* ou clone o repositório
- Instale os pacotes do arquivo *requirements.txt* (Python >=3.10)
- Carregue o arquivo *arduino_code.ino* na sua placa Arduino
- Execute o arquivo *Calibra.Py*
- Siga as instruções nos *widgets* de ajuda
